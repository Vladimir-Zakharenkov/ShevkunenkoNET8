Summary

•	The SportsStore ASP.NET Core project is created using the basic ASP.NET Core
	template.

•	ASP.NET Core has close integration with Entity Framework Core, which is the
	.NET framework for working with relational data.

•	Data can be paginated by including the page number in the request, either using
	the query string or the URL path and using the page when querying the database.

•	The HTML content generated by ASP.NET Core can be styled using popular CSS
	frameworks, such as Bootstrap.
